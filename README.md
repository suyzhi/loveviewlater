# 稍后再看 - 浏览器扩展

在浏览器右侧侧边栏管理稍后再看网页列表，支持右键菜单快速添加，以及浏览进度追踪。

## 功能

- **添加当前页面**：点击侧边栏内「+ 添加当前页」按钮，或右键页面空白处选择「添加到稍后再看」
- **添加链接**：右键任意链接 → 「链接添加到稍后再看」
- **查看列表**：点击工具栏图标打开/关闭右侧边栏，按添加时间倒序排列
- **搜索、筛选、排序**：按标题、网址或来源搜索，并可筛选未读/已读/读到一半/已完成
- **打开页面**：点击列表项，直接跳转到该页面
- **浏览进度追踪**：从侧边栏打开的页面会自动记录滚动百分比，列表项底部显示进度条，并在下次打开时恢复上次滚动位置
- **来源显示**：每条记录显示从哪个网站添加而来，点击来源可打开原来源页面
- **添加反馈**：右键添加成功后，网页内会播放纸片飞入浏览器插件栏方向的动画；侧边栏同步显示轻提示
- **标记已读**：点击复选框给标题添加删除线（左→右滑动动画），再次点击取消
- **永久删除**：点击垃圾桶后变为「删除 / 取消」确认态，确认后才彻底删除；底部「清空全部」一键清空
- **平滑列表动画**：添加、删除、筛选排序和删除确认导致条目移动时，列表会平滑过渡
- **自动去重**：同一 URL 不会重复添加
- **固定底部工具栏**：导出、导入、刷新、清空等按钮始终停留在侧边栏底部
- **手动导出/导入**：导出 JSON 备份或导入恢复数据
- **透明高清图标**：使用网页卡片、书签和时钟组成的透明底扩展图标
- **深色模式**：自动跟随系统主题
- **持久存储**：使用 `chrome.storage.local`，关闭浏览器不丢失
- **一键重载**：修改代码后点击 🔄 按钮即可重新加载扩展，无需前往扩展管理页面

## 安装

1. 打开 Edge/Chrome 浏览器，地址栏输入 `edge://extensions` 或 `chrome://extensions`
2. 开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本项目文件夹

## 技术栈

- Manifest V3
- Side Panel API
- Context Menus API
- Storage API
- Scripting API
