# 稍后再看 - Edge 浏览器扩展

在浏览器右侧侧边栏管理稍后再看网页列表，支持右键菜单快速添加。

## 功能

- **添加当前页面**：点击侧边栏内「+ 添加当前页」按钮，或右键页面空白处选择「添加到稍后再看」
- **添加链接**：右键任意链接 → 「链接添加到稍后再看」
- **查看列表**：点击工具栏图标打开右侧边栏，按添加时间倒序排列
- **打开页面**：点击列表项，在新标签页打开（不干扰当前浏览）
- **删除条目**：悬停后点击 ✕ 删除单条；底部「清空全部」一键清空
- **自动去重**：同一 URL 不会重复添加
- **深色模式**：自动跟随系统主题
- **持久存储**：使用 `chrome.storage.local`，关闭浏览器不丢失

## 安装

1. 打开 Edge 浏览器，地址栏输入 `edge://extensions`
2. 开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本项目 `稍后再看-extension/` 文件夹

## 技术栈

- Manifest V3
- Side Panel API
- Context Menus API
- Storage API
