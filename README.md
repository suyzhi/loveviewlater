# 稍后再看 - 浏览器扩展

在浏览器右侧侧边栏管理稍后再看网页列表，支持右键菜单快速添加，以及浏览进度追踪。

## 功能

- **添加当前页面**：点击侧边栏内「+ 添加当前页」按钮，或右键页面空白处选择「添加到稍后再看」
- **添加链接**：右键任意链接 → 「链接添加到稍后再看」
- **查看列表**：点击工具栏图标打开/关闭右侧边栏，按添加时间倒序排列
- **打开页面**：点击列表项，直接跳转到该页面
- **浏览进度追踪**：从侧边栏打开的页面会自动记录滚动百分比，列表项底部显示进度条
- **标记已读**：悬停后点击 ✕ 给标题添加删除线（左→右滑动动画），再次点击取消
- **永久删除**：右键 ✕ 弹出确认后彻底删除；底部「清空全部」一键清空
- **自动去重**：同一 URL 不会重复添加
- **手动导出/导入**：导出 JSON 备份或导入恢复数据
- **深色模式**：自动跟随系统主题
- **持久存储**：使用 `chrome.storage.local`，关闭浏览器不丢失
- **一键重载**：修改代码后点击 🔄 按钮即可重新加载扩展，无需前往扩展管理页面

## 安装

1. 打开 Edge/Chrome 浏览器，地址栏输入 `edge://extensions` 或 `chrome://extensions`
2. 开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本项目文件夹

## 技术栈

- Manifest V3
- Side Panel API
- Context Menus API
- Storage API
- Scripting API
